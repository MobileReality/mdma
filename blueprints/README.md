# MDMA Blueprints

Ready-to-use interactive MDMA documents for high-stakes domains.

Each blueprint is a complete "outcome-in-a-box": interactive document + demo data + manifest with metrics and requirements.

## Available Blueprints

| Blueprint | Domain | Status |
|-----------|--------|--------|
| `incident-triage` | Critical Ops | Experimental |
| `kyc-case` | Finance/Compliance | Experimental |
| `clinical-ops` | Healthcare | Experimental |
| `customer-escalation` | Customer Ops | Experimental |
| `change-management` | Engineering/Compliance | Experimental |

## Maturity Model

- **experimental** - Basic structure, may have incomplete checklists
- **stable** - Full schema validation, logging, mocks, documentation
- **enterprise-ready** - Security audit, compliance checks, PII handling, full test coverage
